# Battle Droid Game

Консольна гра-симулятор боїв дроїдів на Java.

## Опис

Гра дозволяє створювати різні типи дроїдів, проводити дуелі один на один або командні бої, а також зберігати та відтворювати логи боїв.

## Функціональність

- **Створення дроїдів**: Підтримуються три типи дроїдів:
  - **StrengthDroid** - високий HP (170), середній урон (15-30)
  - **AgilityDroid** - низький HP (95), високий урон (20-55)
  - **UniversalDroid** - збалансовані характеристики (HP: 130, урон: 18-40)

- **Дуель 1×1**: Бій між двома дроїдами з випадковим чергуванням ходів

- **Командний бій**: Бій між двома командами дроїдів

- **Збереження боїв**: Можливість зберегти лог бою у файл

- **Відтворення боїв**: Завантаження та перегляд збережених боїв

## Структура проекту

```
src/game/
├── app/
│   ├── Main.java              # Точка входу
│   └── ConsoleMenu.java       # Консольне меню
├── battle/
│   ├── BattleEngine.java      # Движок для дуелів 1×1
│   ├── BattleLog.java         # Лог бою
│   └── TeamBattleEngine.java  # Движок для командних боїв
├── factory/
│   └── DroidFactory.java      # Фабрика для створення дроїдів
├── model/
│   ├── Droid.java             # Базовий клас дроїда
│   ├── StrengthDroid.java     # Дроїд сили
│   ├── AgilityDroid.java      # Дроїд спритності
│   └── UniversalDroid.java    # Універсальний дроїд
└── storage/
    └── BattleStorage.java     # Збереження/завантаження логів
```

## Вимоги

- Java 17 або вище (використовується switch expressions)

## Запуск

1. Скомпілюйте проект:
```bash
javac -d out/production/Game src/game/**/*.java
```

2. Запустіть програму:
```bash
java -cp out/production/Game game.app.Main
```

## Використання

Після запуску програми ви побачите меню з наступними опціями:

1. **Створити дроїда** - створіть новий дроїд, вибравши тип та вказавши ім'я
2. **Показати список дроїдів** - перегляньте всіх створених дроїдів
3. **Бій 1×1** - проведіть дуель між двома дроїдами
4. **Бій команда×команда** - проведіть командний бій (дроїди автоматично розподіляються на команди)
5. **Записати бій у файл** - збережіть останній проведений бій
6. **Відтворити бій з файлу** - завантажте та перегляньте збережений бій
0. **Вихід** - завершити програму
